# 아이로이드
카센터 물품 및 고객관리용 안드로이드 모바일 어플리케이션.  
각 상품별로 기재된 바코드를 통해 상품을 등록하고  
그 상품을 통한 매출, 입고 관리등을 할 수 있다.  

## 사용기술 및 환경
* Android 1.5
* Java 6
* SQLite
* zxing (지브라 크로싱) 을 통한 바코드 인식기 적용
* 검색 시 자동 완성 기능 적용

## 기타
* 2010년 교내 경시대회 대상
* 2010 경남 e-biz 출품작: 장려상
